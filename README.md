# Infyora Quote Platform

A cloud-native quote generator platform with:

✔ Backend API (Flask)  
✔ Frontend (HTML, JS, CSS, Nginx)  
✔ CI/CD on OpenShift  
✔ GreenOps optimization  
✔ AIOps-ready observability  
✔ Autoscaling using HPA  

## Project Structure
- /backend → Python API
- /front-end → UI served using Nginx
- /infra/openshift → Deployment, Service, Route, HPA

## Tools Used
- **Git & GitHub** for collaboration
- **Docker** for containerization
- **OpenShift** for deployment
- **Knative** (if used)
- **HPA** for auto-scaling (GreenOps)
- **Logging & Metrics** for AIOps

## Features
- Generate motivational quotes
- Auto-scale on traffic
- Nginx-optimized frontend
- Lightweight images (GreenOps best practice)
